Workflows lets you **automate subscription experiences and customer journeys**.

Use workflows to reduce churn, recover failed payments, upsell products, reward loyal customers, and build guided portal experiences.

## Getting started

Workflows follow a simple structure:

**Trigger →** (optional) **Conditions → Actions**

- **Triggers** start a workflow
- **Conditions** decide whether the workflow continues
- **Actions** define what happens next

## Build your first workflow

In the admin:

- Go to **Admin Panel → Workflows**
- Create a workflow and choose a **trigger**
- Add **conditions** and **actions** as needed
- Finish every path with an **End** node
- Use **Preview in Editor** to verify each path
