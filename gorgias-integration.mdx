The Juo and Gorgias integration enables seamless subscription management directly from your Gorgias inbox. You can easily view subscription details, products, upcoming renewals, and recent orders. Additionally, you can skip, reactivate, or cancel subscriptions, with the option to select a cancellation reason.

To activate the Gorgias integration, navigate to **Settings → Integrations** in the Juo merchant portal.

Follow these steps to retrieve your Gorgias API key:

1. Log in to your Gorgias account.
2. Go to **Settings → REST API**.
3. Click the **Create API Key** button.

Next, return to the Juo merchant portal and enter your Gorgias Base API URL, username, and API Key. Click **Save**.
